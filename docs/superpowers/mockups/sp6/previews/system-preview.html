3 · Button hierarchy — one rule sitewide

Today: login + contact submit near-black, booking CTAs clay — same action class, three looks. The rule:

brand — THE action of the surface (book, send, save, pay). One per view.
outline — secondary/dismiss beside a brand action.
ghost — tertiary/inline (table rows, list actions).
destructive — destructive, always behind useConfirm.

The near-black primary variant retires from forms — it stays for neutral chrome only (e.g. "Sign out"). Login/signup/contact submits become brand.
