Editorial, with a calligraphic hand

v1's hands ran too casual — felt-tip, not a brand. This pass keeps the editorial register but adds genuine script: connected calligraphy and high-contrast italic serifs. Nothing here is a marker. Pair every candidate with clipped Public Sans caps for the descriptor so the flourish stays pure signature.

My pick — refined script, not a marker

Cal Barba

Dog Walking & House Sitting

Parisienne. A connected copperplate-style hand — it flows like a real signature but the strokes are modulated and disciplined, so it reads premium where Caveat read cheap. It still feels personal (the whole point for a solo keeper), keeps full legibility down to nav size, and the thin-to-thick contrast rhymes with Fraunces' own high-contrast serifs — the two share a voice. If you want the wordmark to be pure house type instead, Fraunces italic (second bar below) is the on-brand companion — same calligraphic lean, zero new font to load.
